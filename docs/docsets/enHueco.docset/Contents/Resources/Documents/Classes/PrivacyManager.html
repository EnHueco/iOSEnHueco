PrivacyManager

class PrivacyManager

Handles privacy settings

  • turnOffSetting(_:withCompletionHandler:)

    Turns a setting off (e.g. If called as turnOffSetting(.ShowEventsNames), nobody will be able to see the names of the user’s events.

    Declaration

    Swift

  • turnOnSetting(_:for:withCompletionHandler:)

    Turns a setting on for a given policy applied to a group of friends. This replaces the previous setting that the user had.

    If policy nil the setting is applied to everyone

    Declaration

    Swift

    func turnOnSetting(setting: PrivacySetting, `for` policy: PrivacyPolicy? = nil, withCompletionHandler completionHandler: (success: Bool, error: ErrorType?) -> Void)

    Parameters

    setting

    Setting to update
    policy

    Policy applied to the group of friends it accepts for parameter. For example, a method call like turnOnSetting(.ShowNearby, `for`: .EveryoneExcept(aGroup), withCompletionHandler: …) will show to everyone except the members of aGroup that the user is nearby if both indeed are.
